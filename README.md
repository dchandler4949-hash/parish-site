## 🚀 How to Publish (Sync to GitHub)

Firebase Studio stores your files in this editor. To make the website go live, you must "push" these files to your GitHub repository.

### 1. Find the Terminal
Look for the **black box at the bottom of your screen**. 
* If it is hidden, look for a small icon or the word **"Terminal"** at the bottom bar and click it.

### 2. Run these Commands
Copy and paste these lines into that terminal, one by one, pressing Enter after each:

```bash
git add .
git commit -m "Final campaign site updates for 2026"
git remote set-url origin https://github.com/dchandler4949-hash/parish-site.git
git push origin main
```
